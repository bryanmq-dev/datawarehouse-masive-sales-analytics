# Origenes de datos

En esta sección se detallan los origenes de datos implementados para el proyecto, considerando su utilidad y aporte al mismo.

## WordlWideImporters - PostgreSQL

Base de datos Transaccional de nivel Granular con datos de todas y cada una de las ventas realizadas por una empresa de ventas multinacional.

## Brazilian E-Commerce Olist - SQL Server

Base de datos Transaccionla de las ventas realizadas en línea en mercado latinoamericano. Nivel de detalle alto y por cada transacción individual.

## MongoDB Analytics - MongoDB

Base de datos NoSQL de transacciones realizadas por clientes de manera resumida, util para analizar tendencias de ventas centradas en el consumidor.

## DifferentStoreSales - SQLite

Ventas Transaccionales referida a un negocio pequeño (5000 registros) sobre venda a detalle de cada transaccion de usuario.

## SupermarketStoreBranchesSales - SQLite

Ventas resumidas de 896 sucursales distintas de una empresa manufacturera, brindando cantidades totales en forma de reportes.

## RetailStoreSales - SQLite

5000 ventas transaccionales y detalladas de una tienda, donde se pueden analizar detalles más personales de cada cliente.

## Objetivo de estos origenes
Se tomaron en cuenta estos origenes para hacer un análisis desde transacciones granulares y poder realizar comparaciones con otros resumenes de datos.
